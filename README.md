# TechForge PC

TechForge PC is a diploma project developed as part of the Frontend Developer certification program at DIV Academy.

## Project Overview

TechForge PC is a fictional online computer store built as a modern frontend web application. The project demonstrates practical skills in building responsive and interactive user interfaces using current web development technologies.

## Technologies Used

- React
- Tailwind CSS
- JavaScript (ES6+)
- Vite 
- Responsive Web Design principles

## Features

- Product catalog displaying computer hardware and components
- Structured product cards with relevant information
- Category-based organization
- Responsive layout optimized for desktop and mobile devices
- Clean and modern user interface design

## Purpose of the Project

The main objective of this project is to demonstrate frontend development skills, including component-based architecture in React, styling with Tailwind CSS, and building a responsive and user-friendly interface. It also reflects the ability to structure and develop a complete frontend application from concept to implementation.
